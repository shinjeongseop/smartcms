# smartcms

Independent PHP CMS core for reusable member management, level permissions, install wizard, admin pages, and board modules.

## Scope

- Standalone project, independent from `smartboard`
- PHP-based install wizard
- Member login, logout, registration, mypage, profile, password change
- Admin login and protected admin pages
- 1-10 level permission model
- Page and board permissions
- DB-backed board module based on the `oliva1-github` board work
- Access and login logs
